Demo Proxy Ảnh GitHub Pages

Triển khai hệ thống proxy ảnh với 3 phương pháp khác nhau

1. Phương pháp Iframe Embed

Công nghệ sử dụng:
- Iframe sandbox để nhúng nội dung an toàn
- File embed.html xử lý nội dung thực
- Ảnh placeholder giữ URL .jpg cho SEO

2. Phương pháp Service Worker

Công nghệ sử dụng:
- Service Worker intercept request
- Proxy qua jsDelivr CDN
- Cache fallback khi offline
- Giữ nguyên URL ảnh .jpg

3. Phương pháp Hybrid

Công nghệ sử dụng:
- Kết hợp iframe + Service Worker
- 2 lớp fallback (iframe + direct URL)
- Tương thích mọi trình duyệt
- Tối ưu SEO với URL .jpg
